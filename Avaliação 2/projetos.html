Projetos Desenvolvidos

Aqui estão alguns dos projetos que desenvolvi, seja em estudos, trabalhos acadêmicos ou iniciativas pessoais. Eles demonstram minhas habilidades em diferentes tecnologias e áreas do desenvolvimento.

Sistema de Gerenciamento de Tarefas

Um aplicativo web full-stack desenvolvido para auxiliar na organização e acompanhamento de tarefas diárias. Inclui funcionalidades de criação, edição, exclusão e marcação de tarefas como concluídas.
Tecnologias: HTML5, CSS3 (Flexbox), JavaScript (ES6+), Node.js (Express), MongoDB.

Plataforma de E-commerce Básico

Desenvolvimento de um protótipo de e-commerce com catálogo de produtos, carrinho de compras e sistema de checkout simplificado. Foco na experiência do usuário e na arquitetura modular.
Tecnologias: HTML5, CSS3, JavaScript (Manipulação DOM), Python (Flask para backend), SQLite.

Aplicação de Previsão do Tempo

Uma aplicação web que consome uma API externa para exibir a previsão do tempo de diversas cidades ao redor do mundo. Apresenta dados como temperatura, umidade e velocidade do vento de forma intuitiva.
Tecnologias: HTML5, CSS3, JavaScript (Fetch API), Integração com OpenWeather API.
