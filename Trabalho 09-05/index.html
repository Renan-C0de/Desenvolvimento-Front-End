Introdução

Esta página apresenta os conteúdos estudados ao longo das aulas de Desenvolvimento Front-End, com foco em HTML e CSS. A seguir, estão listadas as principais atividades e aprendizados.

Atividade 1 - Introdução ao HTML

Resumo: Aprendemos a estrutura básica de uma página HTML com as tags essenciais como <html>, <head>, <body>, títulos, parágrafos e imagens.

O que aprendi: Compreendi como criar páginas com conteúdo estruturado e semântico usando HTML.

Atividade 2 - Listas, Links e Tabelas

Resumo: Criamos listas ordenadas e não ordenadas, inserimos links com <a> e construímos tabelas com <table>.

O que aprendi: Aprendi como organizar informações em listas e tabelas, além de navegar entre páginas.

Atividade 3 - Formulários em HTML

Resumo: Inserimos campos de entrada, botões, selects e labels utilizando <form>.

O que aprendi: Descobri como criar interfaces para entrada de dados pelo usuário.

Atividade 4 - CSS: Estilos Iniciais

Resumo: Aplicamos cores, tamanhos de fonte, espaçamentos e bordas com CSS direto no HTML e em arquivo externo.

O que aprendi: Percebi como o CSS separa o visual do conteúdo e permite estilizar com flexibilidade.

Atividade 5 - Layouts com CSS

Resumo: Usamos propriedades como float, width e margin para dividir a página em seções como cabeçalho, menu, conteúdo e rodapé.

O que aprendi: Entendi como posicionar elementos de forma organizada e criar layouts simples.

Atividade 6 - Classes e IDs

Resumo: Estilizamos elementos de forma específica com class e id usando seletores no CSS.

O que aprendi: Aprendi a personalizar estilos para diferentes partes da página com mais controle.

Atividade 7 - Cores e Fontes

Resumo: Exploramos diferentes cores de fundo e texto, fontes personalizadas e tamanhos variados.

O que aprendi: Compreendi como melhorar a aparência visual e a legibilidade da página.

Atividade 8 - Box Model

Resumo: Estudamos margin, padding, border e content e como eles afetam o espaço dos elementos.

O que aprendi: Entendi como controlar o espaço interno e externo dos elementos de forma precisa.

Atividade 9 - Responsividade

Resumo: Usamos @media queries para adaptar o site a diferentes tamanhos de tela.

O que aprendi: Descobri como deixar meu site acessível e funcional em celulares e computadores.

Atividade 10 - Tags

Resumo: Entendemos que as tags HTML são elementos utilizados para marcar e estruturar o conteúdo de uma página, como p para parágrafos, h1 a h6 para títulos, para imagens, entre outras.

O que aprendi: Aprendi que as tags são a base da linguagem HTML, funcionando como instruções para o navegador sobre como exibir cada parte do conteúdo.

Atividade 11 - Projeto Final

Resumo: Criamos uma página completa com HTML e CSS baseada no modelo “Portal da Floresta”.

O que aprendi: Pratiquei todos os conceitos de estrutura, estilo, organização e entrega de um site funcional.
